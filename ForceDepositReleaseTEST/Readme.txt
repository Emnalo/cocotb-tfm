Carpetas que contienen los test para cada uno de los tipos de señales.
Cada carpeta esta subdividida en carpetas con el nombre de cada uno de los simuladores que contienen:
	- Ficheros VHDL o VERILOG con la descripción del componente que tratan.
	- Makefile necesario para ejecutar los test y simulador para Cocotb
	- Ficheros Python donde se describen los test ejecutados 
	- Ficheros vcd con las formas de onda del último experimento realizado.
